# 国际化（i18n）

## 概述

**全项目**用户可见文本支持中英文两种语言。覆盖范围包括 TUI 界面、GUI 桌面窗口、CLI 帮助文本、命令错误消息、操作执行输出、Shell 安装提示等所有面向用户的文本。通过 `locale` 配置项切换语言，所有文本集中管理在 `Messages` 结构中。

> 唯一例外：`config.go` 的配置解析错误发生在 locale 解析之前（鸡蛋问题），使用英文硬编码。

## 语言切换

### 配置优先级

```
1. TRY_LOCALE 环境变量
2. ~/.config/try/config.json 中的 locale
3. auto（LC_ALL/LC_MESSAGES/LANG；均空时回退 OS 语言，默认 en）
```

### 有效值

| 值 | 语言 |
|----|------|
| `en` | English |
| `zh` | 中文 |
| `auto` | 自动检测（环境变量以 `zh` 开头为中文；无环境变量时跟 OS 语言） |

### Locale 解析时机

`parseGlobalFlags` 在所有命令处理之前执行，帮助文本的语言由完整优先级链决定（`TRY_LOCALE` > `config.json` > `LANG`）。

GUI 设置对话框支持在运行时切换语言：`SaveLocale` 将选择持久化到 `config.json` 后，重新调用 `i18n.Init` 并重建当前视图，界面文案立即以新语言呈现。

## 语言包

每种语言定义完整的 `Messages` 实例。语言包拆分为三个文件：`messages.go`（`Messages` 结构体与全局 `Init`/`Get` 函数）、`en.go`（英文语言包）、`zh.go`（中文语言包）：

```go
var EN = Messages{...}
var ZH = Messages{...}

func ForLocale(locale string) *Messages
```

`ForLocale` 根据 locale 返回对应语言包（`"zh"` → `ZH`，其他 → `EN`）。

## 消息访问架构

采用进程级全局模式：CLI 入口在 `parseGlobalFlags` 中调用 `i18n.Init(locale)` 设置全局语言包，之后所有模块通过 `i18n.Get()` 访问。

```go
// 初始化（cli/cli.go parseGlobalFlags 中调用一次）
i18n.Init(locale)

// 使用（任意模块）
msgs := i18n.Get()
fmt.Println(msgs.Title)
```

各模块内部通常定义 `func msgs() *i18n.Messages { return i18n.Get() }` 快捷方式，简化调用。

### 例外：配置解析

`config.go` 的解析错误（JSON 语法错误 warning）发生在 locale 解析之前，使用英文硬编码。这是唯一允许的例外。

## Locale 解析

### 完整解析（config 包）

```go
func ResolveLocale(cliLocale string, cfg Config) string
```

优先级：`cliLocale` > `TRY_LOCALE` 环境变量 > `cfg.Locale`。结果为 `"en"` 或 `"zh"` 时直接返回，否则通过 `detectLocale()` 从 LANG 环境变量推断。


## 模块位置

`internal/i18n/` — `messages.go`（Messages 结构、全局 Init/Get）、`en.go`（英文语言包）、`zh.go`（中文语言包）。

## 模块依赖

| 模块 | i18n 使用方式 |
|------|------|
| `internal/i18n/` | Messages 结构、EN/ZH 语言包、Init/Get 全局访问 |
| `internal/config/` | Config.Locale + ResolveLocale（不直接依赖 i18n） |
| `internal/cli/` | `parseGlobalFlags` 中调用 `i18n.Init(locale)` |
| `internal/selector/` | 通过 `i18n.Get()` 访问消息文本 |
| `internal/dialog/` | 通过 `i18n.Get()` 访问消息文本 |
| `internal/script/` | 通过 `i18n.Get()` 访问消息文本 |
| `internal/shell/` | 通过 `i18n.Get()` 访问消息文本 |
| `internal/gui/` | 通过 `i18n.Get()` 访问 GUI 标题、按钮、托盘、Toast、错误提示 |
