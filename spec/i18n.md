# 国际化（i18n）

## 概述

**全项目**用户可见文本支持中英文两种语言。覆盖范围包括 TUI 界面、CLI 帮助文本、命令错误消息、操作执行输出、Shell 安装提示等所有面向用户的文本。通过 `locale` 配置项切换语言，所有文本集中管理在 `Messages` 结构中。

> 唯一例外：`config.go` 的配置解析错误发生在 locale 解析之前（鸡蛋问题），使用英文硬编码。

## 语言切换

### 配置优先级

```
1. --locale 命令行参数（最高优先）
2. TRY_LOCALE 环境变量
3. ~/.config/try/config.json 中的 locale
4. auto（通过 LC_ALL/LC_MESSAGES/LANG 环境变量推断，默认 en）
```

### 有效值

| 值 | 语言 |
|----|------|
| `en` | English |
| `zh` | 中文 |
| `auto` | 自动检测（LC_ALL/LC_MESSAGES/LANG 以 `zh` 开头时为中文，其他为英文） |

### Locale 解析时机

所有命令（包括 `--help` 和 `--version`）统一走完整的全局标志解析流程（`parseGlobalFlags`），不存在轻量解析的例外路径。这确保帮助文本的语言由配置文件和环境变量完整的优先级链决定。

## Messages 结构

集中定义所有用户可见文本，按功能模块分组：

```go
type Messages struct {
    // === selector 主界面 ===
    Title           string
    SearchPrefix    string
    CreateNew       string
    DeleteMode      string // 含 %d
    HintBar         string
    EmptyInputHint  string
    DeleteCancelled string

    // === 删除对话框 ===
    DeleteTitle       string // 含 %d
    DeletePlaceholder string
    DeletePrompt      string
    DeleteFooter      string

    // === 重命名对话框 ===
    RenameTitle  string
    RenamePrompt string
    RenameEmpty  string
    RenameSlash  string
    RenameExists string // 前缀
    RenameFooter string

    // === Ship 对话框 ===
    ShipTitle       string
    ShipDestLabel   string
    ShipMoveLabel   string
    ShipHint        string
    ShipEmptyErr    string
    ShipExistsErr   string // 前缀
    ShipNoParentErr string // 前缀
    ShipFooter      string

    // === CLI 帮助与用法 ===
    HelpText       string // 完整帮助文本（多行）
    UsageClone     string // "Usage: try clone <url> [name]"
    UsageWorktree  string // "Usage: try worktree <dir> [name]"
    UsageDot       string // "Usage: try . <name>"

    // === CLI 错误消息 ===
    ErrParseGitURI string // 前缀 "无法解析 Git URI: "
    ErrParsePath   string // 前缀 "无法解析路径: "
    ErrReadDir     string // 前缀 "无法读取目录: "

    // === 操作执行消息 ===
    MsgCloneFrom     string // "Using git clone to create this trial from %s."
    MsgShipped       string // "Shipped: %s → %s"
    ErrMkdir         string // "创建目录失败"
    ErrMkdirParent   string // "创建父目录失败"
    ErrClone         string // "git clone 失败"
    ErrWorktreeAdd   string // "git worktree add 失败（已创建普通目录）: %v"
    ErrDeletePartial string // "部分删除失败:\n%s"
    ErrRename        string // "重命名失败"
    ErrWorktreeMove  string // "git worktree move 失败"
    ErrMove          string // "移动目录失败"
    ErrUnknownOp     string // "未知的操作类型: %d"

    // === Shell 安装消息 ===
    ErrDetectShell    string // "无法检测 Shell 类型..."
    ErrUnsupportShell string // "不支持的 Shell 类型: %s"
    ErrGetExePath     string // "无法获取 try 可执行文件路径"
    MsgAlreadyInstall string // "try shell integration 已安装在 %s 中。"
    MsgReinstallHint  string // "如需重新安装..."
    ErrCreateDir      string // "创建目录失败"（install 中使用）
    ErrWriteFile      string // "无法写入 %s"
    ErrWrite          string // "写入失败"
    MsgInstalled      string // "已将 try shell integration 写入 %s"
    MsgSourceHint     string // "请运行 source %s 或重启终端以生效。"
}
```

## 语言包

每种语言定义完整的 `Messages` 实例：

```go
var EN = Messages{...}
var ZH = Messages{...}

func ForLocale(locale string) *Messages
```

`ForLocale` 根据 locale 返回对应语言包（`"zh"` → `ZH`，其他 → `EN`）。

## 消息传递架构

### CLI 层（帮助、命令错误）

`Run()` 通过 `parseGlobalFlags` 获取完整的 locale 并创建 `Messages`，之后再处理 `--help` 和 `--version`。`Messages` 存入 `runOptions`，所有 CLI 函数通过 `opts` 访问。

### Selector / Dialog（TUI 界面）

通过 `selector.Config.Messages` 传入 selector，selector 在创建 dialog 时通过 `DialogFactory` 传递给各对话框。

### 操作执行（exec）

`Execute` / `ExecuteTo` 接收 `*Messages` 参数，转发给内部 exec* 函数。导出入口 `ExecClone` / `ExecWorktree` 同样接收 `*Messages`。

### Shell 安装

`Install` 接收 `*Messages` 参数用于安装提示和错误消息。

### 例外：配置解析

`config.go` 的解析错误（JSON 语法错误 warning）发生在 locale 解析之前，使用英文硬编码。这是唯一允许的例外。

## Locale 解析

### 完整解析（config 包）

```go
func ResolveLocale(cliLocale string, cfg Config) string
```

优先级：`cliLocale` > `TRY_LOCALE` 环境变量 > `cfg.Locale`。结果为 `"en"` 或 `"zh"` 时直接返回，否则通过 `detectLocale()` 从 LANG 环境变量推断。


## 模块位置

`internal/i18n/messages.go` — Messages 结构和语言包定义。

## 变更影响

| 模块 | 变更 |
|------|------|
| `internal/i18n/` | Messages 结构扩展（CLI、exec、install 字段） |
| `internal/config/` | Config.Locale + ResolveLocale（不变） |
| `internal/cli/` | runOptions.messages、所有函数使用 Messages |
| `internal/selector/` | Config.Messages（不变），loader 错误消息接入 |
| `internal/dialog/` | 所有对话框接收 Messages（不变） |
| `internal/script/` | Execute/ExecuteTo/ExecClone/ExecWorktree 增加 Messages 参数 |
| `internal/shell/` | Install 增加 Messages 参数 |
