# 国际化（i18n）

## 概述

TUI 界面文本支持中英文两种语言。通过 `locale` 配置项切换语言，所有用户可见的文本字符串集中管理在一个 messages 结构中。

## 语言切换

### 配置优先级

```
1. --locale 命令行参数（最高优先）
2. TRY_LOCALE 环境变量
3. ~/.config/try/config.json 中的 locale
4. auto（通过 LC_ALL/LC_MESSAGES/LANG 环境变量推断，默认 en）
```

### 有效值

| 值 | 语言 |
|----|------|
| `en` | English |
| `zh` | 中文 |
| `auto` | 自动检测（LC_ALL/LC_MESSAGES/LANG 以 `zh` 开头时为中文，其他为英文） |

## 实现方案

### Messages 结构

集中定义所有用户可见文本：

```go
type Messages struct {
    // selector 主界面
    Title           string // 标题
    SearchPrefix    string // 搜索栏前缀
    CreateNew       string // 创建新目录前缀
    DeleteMode      string // 删除模式提示模板（含 %d）
    HintBar         string // 底部快捷键提示
    EmptyInputHint  string // 输入为空时 Ctrl-T 的提示
    DeleteCancelled string // 取消删除提示

    // 删除对话框
    DeleteTitle      string // "Delete %d directories?"
    DeletePlaceholder string // 确认输入 placeholder
    DeletePrompt     string // 确认提示前缀
    DeleteFooter     string // 底部按键提示

    // 重命名对话框
    RenameTitle     string
    RenamePrompt    string // "New name: "
    RenameEmpty     string // 名称为空错误
    RenameSlash     string // 名称含 / 错误
    RenameExists    string // 目录已存在错误前缀
    RenameFooter    string

    // Ship 对话框
    ShipTitle       string
    ShipDestLabel   string // "Destination: "
    ShipMoveLabel   string // "Move to: "
    ShipHint        string // 说明文字
    ShipEmptyErr    string
    ShipExistsErr   string // 前缀
    ShipNoParentErr string // 前缀
    ShipFooter      string
}
```

### 模块位置

`internal/i18n/messages.go` — Messages 结构和语言包定义。

### 消息传递

`Messages` 通过 `selector.Config` 传入 selector，selector 在创建 dialog 时通过 `DialogFactory` 传递给各对话框。

```go
// selector.Config 新增字段
type Config struct {
    // ...existing fields...
    Messages *i18n.Messages
}
```

### Locale 解析

在 `config` 包中添加 `ResolveLocale` 函数：

```go
func ResolveLocale(cliLocale string, cfg Config) string {
    locale := cfg.Locale
    if env := os.Getenv("TRY_LOCALE"); env != "" {
        locale = env
    }
    if cliLocale != "" {
        locale = cliLocale
    }
    switch locale {
    case "en", "zh":
        return locale
    default:
        return detectLocale() // LANG 环境变量推断
    }
}
```

## 变更影响

| 模块 | 变更 |
|------|------|
| `internal/i18n/` | 新包：Messages 结构 + 语言包 |
| `internal/config/` | Config 添加 Locale 字段 + ResolveLocale |
| `internal/selector/` | Config 添加 Messages，view.go/model.go 使用 Messages |
| `internal/dialog/` | 所有对话框接收 Messages 参数 |
| `internal/cli/` | 添加 --locale 标志，构造 Messages 传入 |
